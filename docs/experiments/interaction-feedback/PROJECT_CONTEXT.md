# PROJECT_CONTEXT.md

## このプロジェクトは何か
このプロジェクトは、**LaLaPad Gen2 という左右分割キーボードのトラックパッド周辺に対する改造実験**を扱う。

目的は、既存トラックパッドの操作感を高めるだけでなく、
**マウスを超える機能性・直感性・操作速度**を持つトラックパッドへ発展できるかを検証することにある。

## 外部前提
- 公式ハードウェア組み立ての前提は LaLaPad Gen2 の公式 BuildGuide に従う
- 既存トラックパッドの標準動作は公式 HowtoUse をベースラインとして扱う
- ただし、Codex が主に調査・変更対象とするのは、ユーザーの個人リポジトリ `giniro09/zmk-config-LalaPadGen2`
- 今後の改良はこの個人リポジトリをベースに積み上げる

## プロダクト方向性
既存トラックパッドの周辺に、以下の3系統のフィードバック / 入力を追加し、
LaLaPad Gen2 をより豊かなインタラクションデバイスにできるかを探る。

- ハプティックフィードバック
- 圧力センシング
- 音フィードバック（optional）

## このトラックパッドが目指す価値
- 高級感のあるクリック体験
- タップとは別系統の入力手段としての Force Click
- 強押し中の精密なポインタ操作
- テキスト編集支援（文字カーソル移動）
- 小型トラックパッドでも速く・直感的に使える操作体系

## 優先順位
### 主役として改善したい領域
- ポインタ移動
- クリック
- ドラッグ
- テキスト編集支援

### 特に優先度が高いもの
1. クリック感
2. ポインタ移動の質
3. 強押しによる機能拡張
4. テキスト編集支援

### 後回しでよい領域
- ジェスチャ
- スクロール

## ハプティック / 音 / FSR の役割分担
### ハプティック
- 主役
- click や mode 変化の身体感覚を作る

### 音
- 補助
- 常時は主役にしない
- 実装する場合はデフォルト ON、キーで OFF、状態保持を想定する
- ただし現時点では **Piezo の出力ピンが未確定** のため optional 扱いとする

### FSR
- 単なる閾値スイッチではなく、連続値として活かす
- click の成立
- precision pointer
- caret mode
- 将来の精密制御
に使う

## 検討中の追加ハード
### 確定方向
1. DRV2605L ボード
2. 外付け LRA
3. FSR

### optional
4. Piezo ブザー

### 各パーツを入れる理由
- DRV2605L + LRA:
  触覚的な確認応答を与えるため
- FSR:
  圧力を使った入力 / 疑似 force-touch 的な実験を行うため
- Piezo:
  フル音声系サブシステム無しで、クリック音 / 通知音を出すため
  ただし、現時点では出力ピン未確定のため optional とする

## 既存基板 / ピン前提
ユーザー確認済みの有力ピン候補:
- D1

既存 I2C 使用前提:
- D4 = SDA
- D5 = SCL

現時点の割り当て案:
- D4 / D5: 既存トラックパッド I2C + DRV2605L の共用 I2C
- D1: FSR ADC 入力
- D17: 右側で D1 を空けるための matrix 退避先
- Piezo: **pin pending / optional**

## 現時点の配線前提
### DRV2605L
- 3V3 -> VIN
- GND -> GND
- SDA(D4) -> SDA
- SCL(D5) -> SCL
- IN -> GND
- OUT+ / OUT- -> LRA

### FSR
- 3V3 -> FSR -> node -> D1
- node -> 10kΩ -> GND

### Piezo
- 現時点では未確定
- D16 は使わない
- Piezo を使う場合は、安全な別ピン候補が確認できた後に設計へ戻す

## 重要な機械設計上の気づき
FSR は、静電容量トラックパッドの**表面にそのまま被せる部品**として扱わない。

現時点の理解:
- FSR をタッチ面直上に置くのはリスクが高い
- より良い方向は、トラックパッドや支持構造の**下側の荷重経路**に FSR を入れること
- 「面全体を押せる感触」は、大型 FSR そのものよりも**機械的な荷重集中 / 荷重ルーティング**で作る可能性が高い

## FSR の推奨スタート地点
最初から 4 個の FSR を並べる方針にはしない。

推奨:
- まずは 1 個、必要なら 2 個
- 荷重集中 / 荷重ルーティングを優先
- 面全体の押下感が成立するかを先に確かめる

## 未確定事項 / 実機で確認したいこと
- 正確な LRA 型番選定
- FSR は 1 個で足りるか、2 個必要か
- トラックパッド下で最適な荷重ルーティング位置はどこか
- 1本指 / 2本指 / 3本指の Force 判定がどこまで安定するか
- Piezo に使える安全な出力ピンがあるか
