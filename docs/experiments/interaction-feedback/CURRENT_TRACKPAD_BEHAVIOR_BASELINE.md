# CURRENT_TRACKPAD_BEHAVIOR_BASELINE.md

## このファイルの目的
このファイルは、**現行の LaLaPad Gen2 トラックパッドが何をしているか**を、公式 HowtoUse をもとに短く整理したベースラインです。

今回の改造では、この既存仕様を不用意に壊さずに、
Tap 系の上に Force Click 系と mode 拡張を追加することを目指します。

## 現行の基本動作
- 1本指ドラッグ: カーソル移動
- 2本指ドラッグ: スクロール上下左右
- 1本指タップ: 左クリック
- 2本指タップ: 右クリック
- 3本指タップ: 中クリック
- 2本指ピンチ: ズーム
- 3本指スワイプ: 戻る / 進む / タスクビュー / デスクトップ表示
- TapDrag: タップ後ホールドによるドラッグ継続

## 今回の改造方針との関係
- 既存の Tap 系は残す
- その上に Force Click 系を別入力手段として追加する
- ジェスチャ / スクロールは現段階では主改造対象にしない
- TapDrag と Force Drag は別系統として共存しうる
- 3本指タップ / 3本指 Force は、当面はどちらも middle click 系として扱う
- Piezo は optional であり、使えるピンが確定するまで必須仕様には含めない

## 感度設定
現行仕様では、トラックパッド感度はキー操作で変更でき、設定は電源 OFF 後も保持される。
今回の optional な Piezo ON/OFF なども、実装する場合は同様に保持できると運用上は自然である。
